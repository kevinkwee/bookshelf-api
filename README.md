# Bookshelf API

Bookshelf API is an RESTful API built with Node.JS. This project is my submission for Belajar Membuat Aplikasi Back-End untuk Pemula class at Dicoding.

## Usage
Requires Node.JS >= v14.17.0

```
npm run start
```
